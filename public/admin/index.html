Como a segurança funciona

Este painel e as rotas de gravação ficam atrás do Cloudflare Access. Somente os e-mails definidos na política conseguem entrar. As alterações são gravadas no D1 e as imagens ficam no R2.

Não existe senha escondida no HTML público. O acesso é validado no servidor por um token assinado pelo Cloudflare.

Nenhuma alteração pendente.
